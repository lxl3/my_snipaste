# openSnipaste vs Snipaste 功能差异对照

> 最后更新: 2026-06-01
> 用途: 对照 Snipaste 补全功能缺口，按优先级排序

---

## 图例

- ✅ **已完成** — 功能可用，与 Snipaste 基本持平
- ⚠️ **部分完成** — 功能存在但不够完整

## 我们的存量优势（Snipaste 没有的）

| 功能 | 说明 |
|------|------|
| **Pin 内继续编辑** | Snipaste 的 Pin 只是静态图片，我们的 Pin 可以继续标注/修改 |
| **倒计时覆盖层动画** | 有专门的半透明圆形进度条动画 |
| **Toast 通知系统** | 堆叠式、动画、可悬停暂停 |
| **快捷键帮助面板** | `?` 键触发，显示所有快捷键和工具说明 |
| **MultiHotkey 多快捷键** | 支持多个全局热键（OCR/延时/全屏/固定窗口截图） |
| **快捷键设置冲突检测** | 设置时自动检测并提示冲突 |

## 各文件主要职责（便于快速定位）

| 文件路径 | 负责功能 |
|---------|---------|
| `src/overlay/widget.py` | 截图覆盖层、选区、工具事件分发 |
| `src/overlay/toolbar.py` | 标注工具栏 UI |
| `src/overlay/rendering.py` | 标注渲染到 pixmap |
| `src/overlay/actions.py` | 复制/保存/OCR/钉/撤销重做 |

| `src/core/hotkeys.py` | 全局热键监听 |
